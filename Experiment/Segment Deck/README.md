# Segment Deck

## Purpose
This deck is used to define, manage, and sequence different segments within a radio show or broadcast. It ensures a logical flow and smooth transitions between various parts of the program.

## Key Features
*   **Segment Definition:** Create and name individual show segments (e.g., 'News', 'Interview', 'Music Block').
*   **Segment Sequencing:** Arrange segments in a desired order for the show rundown.
*   **Timing and Duration:** Assign specific durations to each segment and track actual vs. planned time.
*   **Content Association:** Link specific audio, video, or script content to each segment.
*   **Transition Management:** Define and automate transitions between segments.
*   **Segment Status Tracking:** Monitor the live status of each segment (e.g., 'upcoming', 'live', 'completed').
*   **Dynamic Reordering:** Allow for on-the-fly reordering or insertion of segments during a live broadcast.
