# Video Player Deck

## Purpose
This deck is designed to display videos related to the radio show, such as promotional content, behind-the-scenes footage, or visual accompaniments to audio segments. It provides a dedicated interface for managing and playing video content.

## Key Features
*   **Video Playback:** Seamless playback of various video formats.
*   **Playlist Management:** Create and manage video playlists for continuous playback.
*   **Display Integration:** Output video to external displays or streaming platforms.
*   **Synchronization:** Synchronize video playback with audio segments or live broadcast.
*   **Content Library:** Organize and access a library of video assets.
*   **Looping and Autoplay:** Configure videos to loop or autoplay.
*   **Picture-in-Picture (PiP):** Display video in a smaller window while other content is primary.
