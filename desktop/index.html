📋 Overview

Welcome to the Desktop Artifact for the doaashow project. This guide will help you set up, build, and run the desktop application.

What is This?

The desktop artifact contains a complete web-based application optimized for desktop environments. It includes all necessary components, configurations, and build tools to get your application running.

Key Features

  • Modern development workflow with hot reloading
  • TypeScript support for type-safe code
  • HTML5 & CSS3 standards compliant
  • Production-ready build optimization
  • Cross-platform compatibility (Windows, macOS, Linux)
ℹ️ Note: Make sure you have all prerequisites installed before proceeding to the Installation section.

⚙️ Installation

Step 1: Clone the Repository

Clone the doaashow repository to your local machine:

git clone https://github.com/callmecoffee3/doaashow.git
cd doaashow/desktop

Step 2: Install Dependencies

Install all required npm packages:

npm install

Or with yarn:

yarn install
ℹ️ Tip: This may take a few minutes depending on your internet connection. The installation will create a node_modules directory with all dependencies.

Step 3: Build the Application

Build the application for production:

npm run build

Or with yarn:

yarn build
✓ Success: If the build completes without errors, you're ready to run the application!

🚀 Running the Application

Development Mode

Run the application in development mode with hot reloading (changes refresh automatically):

npm run dev

Or with yarn:

yarn dev

The application will typically be available at http://localhost:3000

ℹ️ Tip: In development mode, any changes you save will automatically refresh the application in your browser.

Production Mode

Build and run the production version:

npm run build
npm run start

Or with yarn:

yarn build && yarn start

🔧 Troubleshooting

❌ Issue: Dependencies not installing

Solution: Clear npm cache and reinstall:

npm cache clean --force
rm -rf node_modules package-lock.json
npm install

❌ Issue: Build fails with TypeScript errors

Solution: Update TypeScript:

npm install -D typescript@latest
npm run build

❌ Issue: Port already in use

Solution: Specify a different port:

PORT=3001 npm run dev

❌ Issue: Module not found errors

Solution: Ensure all dependencies are installed:

npm install
npm run build

❌ Issue: Application won't start

Solution: Check if another instance is running and kill it:

# On Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# On macOS/Linux
lsof -i :3000
kill -9 <PID>
