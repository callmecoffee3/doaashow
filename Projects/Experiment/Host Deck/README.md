# Host Deck

## Purpose
This deck provides tools and resources for the show host to manage their on-air presence, interact with guests, and control various show elements. It aims to empower the host with direct control over their broadcast.

## Key Features
*   **Microphone Control:** Easy access to microphone mute, volume, and effects.
*   **Guest Management:** Tools for introducing guests, managing their audio, and facilitating interviews.
*   **Teleprompter/Script Display:** Integrated display for scripts, talking points, and show rundowns.
*   **Audience Interaction:** Tools to view and respond to audience comments, questions, and polls.
*   **Segment Timers:** Visual timers to keep track of segment durations and ensure timely transitions.
*   **Soundboard Access:** Quick access to pre-recorded sound bites, jingles, and applause tracks.
*   **Personalized Settings:** Customizable interface and settings for individual hosts.
