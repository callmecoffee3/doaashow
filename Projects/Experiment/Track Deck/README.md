# Track Deck

## Purpose
This deck is designed for precise management and playback of individual music tracks or pre-recorded audio segments. It offers detailed control over each track for seamless integration into a broadcast.

## Key Features
*   **Individual Track Control:** Start, stop, pause, and cue individual tracks.
*   **Looping and Playback Modes:** Set tracks to loop, play once, or play in sequence.
*   **Volume and EQ per Track:** Adjust audio levels and equalization for each track independently.
*   **Metadata Display:** Show track title, artist, album, and other relevant information.
*   **Waveform Display:** Visual representation of the audio waveform for precise cueing.
*   **Gain Control:** Adjust the input gain for each track to ensure consistent audio levels.
*   **Integration with Music Libraries:** Access and manage tracks from a centralized music library.
